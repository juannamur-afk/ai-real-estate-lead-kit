## Instrucciones de acceso

1. Link de Canva: **[COMPLETAR DESPUÉS DE CREAR LOS DISEÑOS]**
2. Clic en "Usar template"
3. Reemplaza logo, colores y nombre de tu agencia
4. Exporta JPG para posts, PNG para stories

## Paleta de colores sugerida

```
Primario:   #1B4F72  (azul confianza)
Secundario: #F39C12  (naranja acción)
Neutro:     #F8F9FA  (gris claro)
Texto:      #2C3E50  (gris oscuro)
```
